## 实现细节

### 智能播放选择

```dart
// 缓存存在 → 使用缓存
if (await videoFile.exists()) {
  media = Media(cachedPath);
}
// 缓存不存在 → 在线播放 + 后台缓存
else {
  media = Media(videoUrl);
  _silentDownloadVideo(videoUrl);  // 后台
}
```

### 静默下载

```dart
// 不显示进度，不阻塞 UI
Future<void> _silentDownloadVideo(String videoUrl) async {
  try {
    await VideoCacheManager.getOrDownloadVideo(videoUrl);
  } catch (e) {
    // 失败时静默处理，不影响在线播放
  }
}
```

## 常见问题速答

**Q: 首次为什么不等缓存完成？**
A: 为了最优的用户体验。在线播放 + 后台缓存，二者最佳组合。

**Q: 缓存下载失败怎么办？**
A: 不影响，已在线播放了。再试一次会重新缓存。

**Q: 离线能看吗？**
A: 已缓存的视频可以离线看。

**Q: 怎么禁用缓存？**
A: 设置中清理缓存即可。
